# Data-Analysis<br>
DATA ANALYSIS ON NETFLIX MOVIES AND TV SHOWS<br>
This project presents a comprehensive data analysis of Netflix Movies and TV Shows using a publicly available Kaggle dataset. The analysis is conducted in Python within a Jupyter Notebook environment, leveraging key libraries such as Pandas and NumPy.

Dataset Overview:

Contains 8,807 rows and 12 columns, totaling 105,684 data points.

Includes features such as: Title, Type (Movie/TV Show), Director, Cast, Country, Date Added, Release Year, Rating, Duration, Genres (Listed In), and Description.

Missing values were identified in several columns: Director (2,634), Cast (825), Country (831), Date Added (10), Rating (4), and Duration (3). These null values were handled using the .fillna() method in Pandas.

Technologies Used:
Python, Pandas, NumPy, Jupyter Notebook.

This analysis explores data cleaning, preprocessing, and insight generation based on Netflix content.
